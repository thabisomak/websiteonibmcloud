Hosting A Small Website on the IBM Cloud

Thabiso Makhopane | 15 December 2020

In this post, I’ll be sharing my experience deploying and hosting a small business website on the IBM cloud. I set about trying to figure out what the best combination of services would be and found 2 solutions that I liked.

Option 1) To deploy a WordPress site onto a Kubernetes cluster using Helm chart. Seeing as how WordPress is the most popular CMS in the world, it made sense to explore this option as it would appeal to many web developers (Myself included).

Or;

Option 2) To deploy a static website using IBM’s Cloud Object Storage (COS) and implementing the application backend using IBM Cloud Functions. Since lots of people prefer static sites for their speed and security, this was also a viable option.

Well….turns out I’m painfully indecisive as I decided to try both.
